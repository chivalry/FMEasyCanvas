FMEasyCanvas Change Log

Beta 1 (10-27-2014)
• Initial public beta.
• No support for Windows.
• Removed support for image library.

Beta RC2 "Division" (10-26-2014)
• Added support for additional settings.
• Performance enhancements - dropped single Let statement approach in WV.
• Redesigned token handling.
• Split consolidated file into components and stored in globals.
• Completed demo database, with iPad-friendly theme.

Beta RC1 "Entropy" (10-25-2014)
• Fully consolidated file, including jQuery, React, Literally Canvas, CSS.
• Customizations to LC to load some tool images (dashed-line.png, line-with-arrow.png) as inline Base64-encoded values.
• Tokenized HTML w/ full integration into FM via WebViewer.
• Added support for a number of settings (watermark, save button, body CSS, etc).
• Token substitutions are in place.

Alpha 4 (10-24-2014)
• First attempt at consolidated file.
• Tool PNG files added to code file as B64.
• Debug code added in attempt to add support for Win IE.

Alpha 3 (10-10-2014)
• Additional improvements to the demo database.
• Full support for FMP URL callbacks via "Export Canvas" event handler.

Alpha 2 (10-09-2014)
• Additional Literally Canvas init features supported.
• First attempt at basic image library support.
• Added support for background images.
• Added support for watermarks.
• Demo database improved, but all HTML, JS, etc remain external.

Alpha 1 (10-01-2014)
• Alpha release to small test group.
• Only basic functionality implemented.
• Initial build w/ externally referenced HTML, jQuery, JS libraries, etc.
• No support for Windows.
