FMEasyCanvas: Frequently Asked Questions

What is FM EasyCanvas?
EasyCanvas is an open source drawing solution for the FileMaker platform. With EasyCanvas, you can easily add drawing functionality to your FileMaker solutions. For example, you can create drawings from scratch, or make changes to existing drawings or photos, and then save the drawings to container fields in your database.

What does the canvas look like?
Click here to see a screenshot showing EasyCanvas's UI.

What versions of FileMaker does EasyCanvas support?
As of Beta 1, EasyCanvas supports users running FileMaker Pro on Mac OS X, and users of FileMaker Go. We are working on support for Windows-based users.

Is EasyCanvas compatible with FileMaker WebDirect?
As of this writing, no. We are working on support for WebDirect.

Does EasyCanvas require plugins?
No plugins are needed.

Does EasyCanvas support customization?
Yes. There are several options available for customizing EasyCanvas. For example, you can easily control the initial background color of the canvas, the tools that are available to users, and more. For more information, refer to the Settings page.

How much will EasyCanvas cost?
EasyCanvas is being provided via an open source license.

What technologies is EasyCanvas based on?
EasySync is based on standards that include HTML, CSS, Javascript / jQuery, as well as the following libraries: jQuery, React, and Literally Canvas.

Who developed EasyCanvas?
EasyCanvas was developed by Tim Dietrich, a database consultant that specializes in the FileMaker platform.
