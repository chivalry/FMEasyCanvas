FMEasyCanvas: Open Issues List

Here are the known and open issues that we are working on.

No support for Windows users.
FMEasyCanvas does not support FileMaker Pro running on Windows.

Limited support for FileMaker WebDirect.
At this time, FMEasyCanvas will display properly in WebDirect, but there is no support for saving images. This is because WebDirect does not support FMP URLs.

When saving marked-up images, only changes to the canvas area that includes changes is included.
