FMEasyCanvas: Settings

The following global variables can be used to customize EasyCanvas. For example of these settings being used, see the demo database.

ezc_add_on_div_style
• CSS used in the DIV area of the "Add-On" area, which wraps the "save" button.
• Default: background: "#ffffff; padding-left: 10px; padding- right: 10px;"

ezc_add_on_form_style
• CSS used in the "Add-On" form, where the "save" button is located.
• Default: "margin: 0; padding: 0;"

ezc_body_style
• CSS for the BODY tag.
• Default: "margin: 0; padding: 0; background: #000"

ezc_canvas_bgcolor
• Initial background color used for the canvas.
• Default: "transparent"

ezc_canvas_bgimage
• Base64-encoded image to be placed in the background of the canvas.
• This image will not affected by drawing, erasing, clearing, etc.
• Default: null

ezc_canvas_bgimage_x
• X coordinate for the background canvas image.
• Default: 0

ezc_canvas_bgimage_y
• Y coordinate for the background canvas image.
• Default: 0

ezc_canvas_height
• Height, in pixels, of the canvas. Set to NULL for an infinitely sized canvas.
• Default: null

ezc_canvas_width
• Width, in pixels, of the canvas. Set to NULL for an infinitely sized canvas.
• Default: null

ezc_export_type
• The file type to use when exporting the canvas.
• Options supported include "png" and "jpeg."
• Default: "png"

ezc_fill_color
• Initial fill color.
• Default: "#fff"

ezc_fmprul_save
• The URL to use when the user chooses to save the image.
• Should be in the form of an FMP URL.
• Default: "fmp://$/FMEasyCanvas.fmp12?script=SaveIt¶m="

ezc_initial_image
• An image that will be placed on the canvas for editing.
• Default: null

ezc_initial_image_scale
• An optional scale for the initial image.
• Default: 1

ezc_initial_image_x
• X coordinate for the initial canvas image.
• Default: 0

ezc_initial_image_y
• Y coordinate for the initial canvas image.
• Default: 0

ezc_save_css
• CSS used to style the "save" button.
• Default: "width: 300px; margin-top: 15px; margin-bottom: 10px; padding: 8px; font-family: Tahoma; border-radius: 5px; background: #eeeeee; color: blue; font-size: 12pt;"

ezc_save_text
• Text used for the "save" button.
• Default: "Save Canvas to FileMaker"

ezc_stroke_color
• Initial stroke color.
• Default: "#000"; ezc_stroke_color

ezc_tools
• The tools that will be provided to the user.
• Tools are loaded in the order listed.
• Tools should be comma-separated.
• Default: "Pencil, Eraser, Line, Text, Rectangle, Ellipse, Pan, Eyedropper"

ezc_watermark_image
• An image to be display behind the drawing, for branding purposes.
• Like the background image, it will not affected by drawing, erasing, clearing, etc.
• Unlike the background image, this image is always centered and is not affected by the pan tool.
• Must be a Base64-encoded image.
• Default: null
